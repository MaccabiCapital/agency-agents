# Runbook: SEO Authority Campaign

**Scenario**: Execute a focused local SEO authority push for a client — building visibility, GBP strength, citation consistency, and content to move rankings in a target market.

**Typical timeline**: 4–8 weeks for meaningful movement; ongoing for sustained results
**Agents**: Marketing Lead (lead), Web Lead (on-page fixes), Support Lead (review generation)
**Deployment mode**: Micro Task or Build Sprint depending on scope

---

## When to Use This Runbook

- Client ranks below top 3 in local pack for primary service + city
- Client has a weak GBP profile (few reviews, incomplete, no posts)
- Client has citation inconsistencies holding back rankings
- Client has a solid site but low organic traffic
- Post-launch campaign to build authority after a new website goes live

## Campaign Structure

### Week 1 — Audit & Baseline

- [ ] Run full SEO audit (if not done in Phase 0) — use phase-0 checklist
- [ ] Record baseline rankings for top 5 target keywords (use SerpWatcher, BrightLocal, or manual search)
- [ ] Record GBP baseline: views, calls, direction requests (last 28 days)
- [ ] Identify top 3 local competitors and their GBP review counts
- [ ] Identify citation gaps: check top 10 directories for NAP accuracy
- [ ] Identify on-page gaps: pages missing from site, thin content, missing schema

### Weeks 2–3 — Foundation Fixes

Priority order: fix what's actively hurting first.

**On-page (Web Lead)**
- [ ] Fix any title tags or meta descriptions that are missing or duplicated
- [ ] Add LocalBusiness + Service schema to all key pages
- [ ] Create any missing service pages or location pages in scope
- [ ] Optimize H1s and first 100 words of content for primary keyword + city

**GBP (Marketing Lead)**
- [ ] Update categories (primary + secondary) if suboptimal
- [ ] Complete business description (750 chars, keyword-relevant)
- [ ] Upload 10+ photos if below that threshold
- [ ] Publish weekly GBP posts (service, offer, or tip format)
- [ ] Answer any unanswered Q&As

**Citations (Marketing Lead)**
- [ ] Fix NAP on top 5 directories: Google, Yelp, Bing, Facebook, Healthgrades/HomeAdvisor (vertical-appropriate)
- [ ] Submit to any missing tier-1 directories
- [ ] Flag suppressed or duplicate listings for removal

**Reviews (Support Lead)**
- [ ] Confirm review request automation is active and firing
- [ ] If no automation: implement manual review request process
- [ ] Respond to all existing reviews that have no response
- [ ] Target: 4+ new reviews per month during campaign

### Weeks 4–8 — Content & Authority

- [ ] Publish 1–2 service or FAQ pages targeting secondary keywords
- [ ] Publish 2 GBP posts per week (sustained)
- [ ] Build 5–10 additional citation submissions (tier-2 directories)
- [ ] Monitor rankings weekly — flag any movement (positive or negative)
- [ ] If rankings stagnate after week 4: reassess competitor gap and adjust content angle
