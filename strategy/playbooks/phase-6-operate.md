# Phase 6 — Operate

**Objective**: Keep the client's systems performing, generating evidence of value, and give them reasons to stay. Identify improvement opportunities before they ask.

**Agents**: Support Lead (lead), Marketing Lead (SEO + content), Automation Lead (optimization), Operations Lead (reporting)
**Duration**: Ongoing — monthly recurring cycle
**Output**: Monthly report, review velocity, optimization actions, SOP updates

---

## Inputs Required Before Starting

- [ ] Phase 5 go-live handoff complete
- [ ] GA4, GSC, GBP, and GHL access confirmed for reporting
- [ ] Baseline metrics recorded at launch
- [ ] Client communication preferences confirmed (email, GHL, call cadence)

---

## Monthly Operating Cycle

### Week 1 — Data Collection & Review

**Web & SEO**
- [ ] Pull GA4: sessions, users, top pages, organic vs. other sources (compare to prior month and launch baseline)
- [ ] Pull GSC: impressions, clicks, average position for top queries
- [ ] Run Lighthouse on homepage — flag if mobile score dropped below 75
- [ ] Check for manual actions or coverage issues in GSC

**GBP**
- [ ] Pull GBP Insights: views, calls, direction requests, photo views
- [ ] Check for new Q&As to answer
- [ ] Publish GBP post for the month (service highlight, offer, or tip)
- [ ] Review any new reviews — flag if negative reviews need response

**GHL / Pipeline**
- [ ] Pull pipeline report: new leads, contacted, booked, won, lost this month
- [ ] Check automation health: any errors, failed sends, or dead sequences?
- [ ] Review speed-to-lead performance: are leads being contacted within 5 minutes?
- [ ] Review review generation: how many requests sent, how many reviews received?

---

### Week 2 — Optimization Actions

Based on data review, execute the highest-priority improvement:

**If organic traffic is flat or declining**
- [ ] Check for ranking drops in GSC — identify affected pages
- [ ] Review competitor GBP changes (new reviews, new posts, category changes)
- [ ] Update on-page content on underperforming pages
- [ ] Publish new GBP posts / update GBP description

**If pipeline conversion is low**
- [ ] Review lead source quality — are the right leads coming in?
- [ ] Check speed-to-lead automation — is it actually firing?
- [ ] Update follow-up sequence copy if open/response rates are low
- [ ] Consider adding a booking offer or incentive to the nurture sequence

**If reviews are not growing**
- [ ] Verify review request automation is firing after appointments
- [ ] Audit GHL: are appointments being marked complete?
- [ ] If manual process: coach client on asking for reviews post-visit
- [ ] Publish responses to all reviews older than 7 days with no response

**Site performance issue**
- [ ] Identify cause (new images, plugin update, third-party script)
- [ ] Fix and re-test — document fix in SOP log

---

### Week 3 — Reporting

Compile the monthly client report. See activation prompt for reporting format.

Report covers:
- Lead volume and pipeline activity
- Website traffic trends
- GBP performance
- Reviews: new count, total, avg rating
- Top win of the month (one clear result to highlight)
- Next month priority: one focused recommendation

**Delivery**
- [ ] Report formatted (doc, PDF, or GHL dashboard — per client preference)
- [ ] Sent to client by [agreed date each month]
- [ ] Follow-up call scheduled if report shows significant changes (good or bad)

---

### Week 4 — Retention & Forward Planning

**Client health check**
- [ ] Is client using GHL? (log-in data or ask directly)
- [ ] Any unresolved complaints or support requests?
- [ ] Upcoming events, promotions, or seasonal needs to plan for?
- [ ] Flag if client has not responded to 2+ reports — proactive check-in call

**Upsell identification**
Flag to Strategy Lead if client shows any of:
- [ ] Strong lead volume but low conversion → suggest pipeline/sales coaching
- [ ] Growing traffic but no reviews → suggest reputation campaign
- [ ] Good reviews but flat traffic → suggest content or authority campaign
- [ ] Seasonal business approaching peak → suggest paid traffic or promotion

**SOP Updates**
- [ ] Did anything unusual happen this month? Document it.
- [ ] Did a new automation or content format work well? Add to playbook.
- [ ] Did a revision cycle fail? Note the root cause.

---

## Quality Gate — Monthly Operate Review

Each month, Operations Lead confirms:
- [ ] Report sent on time
- [ ] At least one optimization action completed
- [ ] No open client issues older than 48 hours
- [ ] Automation health check done (no errors)
- [ ] GBP post published
- [ ] Next month's priority identified and logged
