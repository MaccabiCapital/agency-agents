# Phase 0 — Client Discovery & Audit

**Objective**: Build a complete picture of the client's current state before recommending anything. No scope, no strategy, no proposals until discovery is done.

**Agents**: Marketing Lead (lead), Web Lead (support), Strategy Lead (review)
**Duration**: 1–3 days
**Output**: Discovery audit doc with prioritized findings, quick wins flagged

---

## Inputs Required Before Starting

- [ ] Client business name, URL, and primary location(s)
- [ ] Business type / vertical (healthcare, dental, chiro, home services, etc.)
- [ ] Client's stated problems or goals (however vague)
- [ ] Access to Google Business Profile (if available)
- [ ] Access to existing analytics or GSC (if available — not required)

---

## Step 1: Website Audit

Marketing Lead or Web Lead runs this. Document everything — do not filter yet.

**Technical**
- [ ] Page speed: run Lighthouse on homepage (mobile + desktop), record scores
- [ ] Mobile display: view on 375px — does content render correctly?
- [ ] HTTPS confirmed, no mixed content warnings
- [ ] Core Web Vitals: note LCP, CLS, FID/INP from PageSpeed Insights

**On-Page SEO**
- [ ] Title tags: present, unique, keyword-relevant?
- [ ] Meta descriptions: present, compelling?
- [ ] H1 on every key page: present, accurate?
- [ ] Service pages: do they exist? Are they indexed?
- [ ] Location data: city/state present on homepage and contact page?
- [ ] Schema markup: LocalBusiness schema present and valid?

**Trust & Conversion**
- [ ] Primary CTA visible above the fold on mobile?
- [ ] Phone number clickable on mobile?
- [ ] Reviews/testimonials present?
- [ ] Credentials, licenses, or certifications shown?
- [ ] Contact form working? Test submit.

---

## Step 2: Local SEO Audit

**Google Business Profile**
- [ ] GBP claimed and verified?
- [ ] Primary category correct?
- [ ] Description complete (750 chars)?
- [ ] Hours accurate?
- [ ] Photos: minimum 10 uploaded?
- [ ] Reviews: count, average rating, response rate
- [ ] Posts: last post date

**Citations & NAP**
- [ ] Check NAP (Name, Address, Phone) on top 5 directories: Google, Yelp, Bing, Facebook, Healthgrades (if healthcare)
- [ ] Any inconsistencies? Note exact discrepancies.

**Competitive Snapshot**
- [ ] Search "[primary service] + [city]" — who ranks top 3?
- [ ] Note GBP review counts and star ratings for top 3 competitors
- [ ] Identify one gap the client could exploit (fewer reviews needed, underserved keyword, etc.)

---

## Step 3: Pipeline & Follow-Up Assessment

Gather from client intake conversation or intake form. Document what actually exists.

- [ ] How are leads currently coming in? (calls, forms, walk-ins, referrals)
- [ ] What happens when a form is submitted? (manual, automated, unknown)
- [ ] How fast does someone follow up? (same day, next day, unknown)
- [ ] Is there a CRM in use? (GHL, other, none)
- [ ] Is there an online booking option?
- [ ] How are reviews currently generated? (asked manually, automated, not at all)

---

## Step 4: Compile Findings

Structure the audit doc as:

1. **At a Glance** — 3–5 bullet summary of biggest issues
2. **Website Findings** — Technical, SEO, conversion — by severity (Critical / High / Medium)
3. **Local SEO Findings** — GBP, citations, competitive gaps
4. **Pipeline Findings** — Current lead flow, follow-up gaps
5. **Quick Wins** — 3–5 actions that could show results within 30 days
6. **Recommended Scope** — What Phase 1 strategy should address first

---

## Quality Gate: Phase 0 → Phase 1

PASS requires:
- [ ] All three audit sections completed
- [ ] Lighthouse scores recorded
- [ ] At least one test form submission attempted
- [ ] GBP status confirmed
- [ ] Quick wins list included
- [ ] Strategy Lead has reviewed and approved audit doc

Do not schedule Phase 1 until this gate is cleared.
