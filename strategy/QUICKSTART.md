# RescueMyWorkday — Quick-Start Guide

> Get from zero to active client pipeline in 5 minutes.

---

## What This System Does

RescueMyWorkday runs AI-assisted delivery for small service businesses. The operating model is **Acquire → Convert → Deliver → Retain**. These docs tell every agent in the system what to do, in what order, and how quality is verified before anything moves forward.

## New Client: Full Engagement

```
Activate Strategy Lead for new client onboarding.

Client: [CLIENT NAME]
Business type: [healthcare / local service / other]
Known problems: [website / SEO / pipeline / follow-up / all]

Run full engagement sequence:
- Phase 0: Discovery audit (website, SEO, GBP, pipeline review)
- Phase 1: Strategy doc and scope agreement
- Phase 2: Foundation (GHL sub-account, domain, hosting)
- Phase 3: Build (website, automations, content, integrations)
- Phase 4: QA and client preview
- Phase 5: Go-live and handoff
- Phase 6: Operate (monthly reporting, reviews, optimization)

Quality gate required between every phase. Evidence required for all QA passes.
```

## Key Concepts

1. **Quality Gates** — No phase advances without evidence-based sign-off
2. **ACDR Loop** — Acquire, Convert, Deliver, Retain. Every task maps to one phase
3. **Handoffs** — Structured context transfer between agents; never start cold
4. **Evidence Standard** — Screenshots, test results, Loom walkthroughs — not claims
5. **GHL First** — GoHighLevel is the primary CRM/automation platform for all clients
6. **Max 2 Revision Cycles** — After two rounds of rework, escalate to lead
